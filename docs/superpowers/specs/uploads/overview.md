## What It Is

Anthropic's **first official technical credential**, launched March 12, 2026. A proctored, architecture-level exam validating the ability to design and ship **production-grade Claude AI applications at enterprise scale**.

This is not a conceptual primer or click-through quiz. It tests **systems-level thinking** — fallback loop design, Batch API cost optimization, JSON schema structuring, MCP tool orchestration. It's strictly proctored with no access to Claude, external tools, or documentation during the exam.

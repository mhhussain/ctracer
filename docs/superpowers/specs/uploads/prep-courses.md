## Course 3: Introduction to Model Context Protocol
**Link:** anthropic.skilljar.com/introduction-to-model-context-protocol  
**Level:** Intermediate | **Exam domain:** 4 (Tool Design & MCP)

### Modules & Lessons
**Introduction**
- Welcome + introducing MCP
- MCP clients

**Hands-on with MCP Servers**
- Project setup
- Defining tools with MCP
- The Server Inspector

**Connecting with MCP Clients**
- Implementing a client
- Defining resources / accessing resources
- Defining prompts / prompts in the client

**Assessment and Wrap Up**
- Final assessment
- MCP review

### Key Concepts
- MCP shifts tool definition/execution burden from app servers to specialized MCP servers
- Three primitives: Tools (model-controlled), Resources (app-controlled), Prompts (user-controlled)
- Use Python SDK decorators instead of manual JSON schemas
- Static vs. templated resources
- Browser-based Server Inspector for testing

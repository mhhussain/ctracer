# UI Polish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix visual regressions across 9 screens/components — all CSS already exists in `index.css`; every fix is a JSX structural correction only.

**Architecture:** Each task is a single self-contained JSX file. Tasks do not depend on each other and must be committed individually. No CSS changes. No data changes. No hook changes.

**Tech Stack:** React 18, Vite, `react-router-dom` v6, `web/src/index.css` (design system), `web/src/components/Card.jsx` (border/bg/padding wrapper).

**Branch:** Create `fix/ui-polish` from `feature/web-app-completion` before starting.

**Dev server:** `cd web && npm run dev` — runs at http://localhost:5173. Use it to visually verify each task.

**Design ground truth:** `docs/superpowers/specs/2026-05-27-ui-polish-design.md` — the authoritative spec with BEFORE/AFTER for every change.
